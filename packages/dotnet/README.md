# .NET SDK

The Beta-1 .NET SDK is distributed as the public NuGet package:

- `Foxlin.Aegir.Security`
- version `0.1.0-Beta.1`

This repository keeps the public-facing package documentation and example consumer here, while the package itself is installed from NuGet.

## Install

```powershell
dotnet add package Foxlin.Aegir.Security --version 0.1.0-Beta.1
```
